此網站作為 API call invocation reporting system 的互動界面，提供
使用者查看資料庫內容的互動界面，以及根據基於 Temu 產生的
all_API call execution traces 來讓使用者搜尋用到特定 DLL/API 的
samples，並針對 traces 統計後列出最常被 load 的 dll ranking
  • API info page

    搜索資料庫中 API 的資訊

  • Dll info page

    搜索資料庫中 DLL 的資訊

  • Search page

    根據所指定的 DLL/API 列出有使用到該 DLL/API 的 samples

  • Frequency page

    根據資料庫中所蒐集的 sample 的 all_API call execution traces
    進行統計，依照被 load 的 dll 的頻率進行 ranking
